Simulacin TP3: Clculo de Prdidas de Calor

En este notebook calculamos la prdida de calor del calentador elctrico de agua con resistencia de NICROM.

In[1]:
import numpy as np
import matplotlib.pyplot as plt
from math import pi

Parmetros del Calentador

In[2]:
# Parmetros geomtricos
DIAMETRO = 8.0  # cm
ALTURA = 15.0   # cm
RADIO = DIAMETRO / 2  # cm
VOLUMEN = pi * (RADIO**2) * ALTURA  # cm^3
CAPACIDAD = VOLUMEN / 1000  # litros

# Parmetros del aislante
ESPESOR_AISLANTE = 0.25  # cm
COEF_CONDUCTIVIDAD_TERMICA = 0.04  # W/(mK) - Fibra de vidrio

# Parmetros elctricos (del TP1)
VOLTAJE = 12.0  # V
RESISTENCIA = 0.23  # Ohms
POTENCIA = VOLTAJE**2 / RESISTENCIA  # W

# Parmetros del fluido
DENSIDAD_AGUA = 1.0  # kg/L
MASA_AGUA = CAPACIDAD * DENSIDAD_AGUA  # kg
CALOR_ESPECIFICO_AGUA = 4180  # J/(kgC)
TEMP_INICIAL = 20.0  # C
TEMP_AMBIENTE = 20.0  # C
TEMP_OBJETIVO = 80.0  # C

Clculo de la Superficie y Prdida de Calor

In[3]:
# Conversin a metros para los clculos
RADIO_M = RADIO / 100  # m
ALTURA_M = ALTURA / 100  # m
ESPESOR_M = ESPESOR_AISLANTE / 100  # m

# Clculo de la superficie
SUP_LATERAL = 2 * pi * RADIO_M * ALTURA_M  # m^2
SUP_BASES = 2 * pi * RADIO_M**2  # m^2
SUP_TOTAL = SUP_LATERAL + SUP_BASES  # m^2

# Clculo de la prdida de calor
PERDIDA_CALOR = COEF_CONDUCTIVIDAD_TERMICA * SUP_TOTAL / ESPESOR_M  # W/K

# Mostrar resultados
print(f"--- Dimensiones del Calentador ---")
print(f"Dimetro: {DIAMETRO} cm")
print(f"Altura: {ALTURA} cm")
print(f"Capacidad: {CAPACIDAD:.2f} litros")

print(f"\n--- Clculo de Superficie ---")
print(f"Superficie lateral: {SUP_LATERAL:.6f} m")
print(f"Superficie de las bases: {SUP_BASES:.6f} m")
print(f"Superficie total: {SUP_TOTAL:.6f} m")

print(f"\n--- Prdida de Calor ---")
print(f"Coeficiente de conductividad trmica (fibra de vidrio): {COEF_CONDUCTIVIDAD_TERMICA} W/(mK)")
print(f"Espesor del aislante: {ESPESOR_AISLANTE} cm")
print(f"Prdida de calor: {PERDIDA_CALOR:.3f} W/K")

Anlisis de Prdida vs. Diferencia de Temperatura

In[4]:
# Crear un rango de diferencias de temperatura
dif_temperatura = np.linspace(0, 60, 13)  # 0C a 60C en 12 pasos (cada 5C)

perdida_en_watts = PERDIDA_CALOR * dif_temperatura

# Calcular el porcentaje de la potencia total
porcentaje_perdida = (perdida_en_watts / POTENCIA) * 100

Conclusiones

El calentador diseado con las especificaciones dadas presenta una prdida de calor de aproximadamente 0.77 W/K. Esto significa que:

  1. A mayor diferencia entre la temperatura del agua y la temperatura ambiente, mayor ser la prdida de calor.
  2. En el peor caso (diferencia de 60C), la prdida representa aproximadamente el 7.4% de la potencia disponible.
  3. Estas prdidas afectarn el tiempo total necesario para calentar el agua hasta la temperatura objetivo.
